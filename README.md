# Editorial System — що отримує користувач

> Перелік функцій платформи з точки зору людей, які нею користуватимуться.
> ✅ — готово до використання. 🟢 — входить у MVP, буде готове до релізу. ⛔ — за окремою домовленістю.

---

|---|---|

| Візуальна шкала 20 кроків: бачу, на якому етапі мій рукопис зараз і що попереду | 🟢 |

| Отримую зауваження Рецензента → завантажую виправлений варіант | 🟢 |

| Узгоджую фінальні метадані перед публікацією | 🟢 |
| Бачу всі дедлайни зі своєї сторони і отримую нагадування заздалегідь | 🟢 |

## Що буде «на виріст» (за окремою домовленістю)

- **Кабінет Фінансів** — облік рахунків, звіти по оплатах
- **Кабінет Юриста** — керування договорами, e-signature (Вчасно / Diia.Signature)
- **Онлайн-оплата** через LiqPay / Fondy / Stripe / WayForPay / iPay
- **Автоматичні перевірки** на плагіат (iThenticate) і AI-контент (GPTZero, Turnitin)
- **Багатомовний інтерфейс** (англійська, польська)
- **Повнотекстовий пошук** по всіх рукописах і файлах
- **Аналітика** — звіти по кількості публікацій, середньому часу стадій, SLA-метрикам
- **Експорт** у CSV / PDF / DOCX
- **Мобільні застосунки** iOS та Android
- **Публічний каталог** опублікованих творів
- **Внутрішній чат / коментарі** між учасниками одного рукопису
