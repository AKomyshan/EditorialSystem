## Поза обсягом MVP (окремі договори)

- **Кабінет Finance** — керування рахунками, інтеграція з бухгалтерією
- **Кабінет Legal** — керування договорами, e-signature інтеграція
- **Реальні інтеграції**:
  - e-signature (DocuSign / Вчасно / Diia.Signature)
  - платіжні шлюзи (LiqPay / Stripe / Fondy)
  - автоматичні плагіат-чеки (iThenticate / Unicheck)
  - AI-детектори (GPTZero / Turnitin AI)
- **Мультимовність UI** (EN, PL) — зараз тільки українська
- **Повнотекстовий пошук** по рукописах і файлах
- **Аналітика та звіти** (кількість публікацій, час на стадію, SLA-метрики)
- **Експорт** у CSV / PDF / DOCX
- **Mobile app** (iOS / Android)
- **Публічний каталог** опублікованих творів
- **Форум / чат між учасниками** одного рукопису
- **API для зовнішніх інтеграцій** (OAuth2 клієнти, вебхуки)
